help_outline How to Play
1

Reveal a Tile

Left-click any tile to reveal it. The first click is always safe — mines are placed after it.

2

Read the Numbers

Each number tells you how many mines are in the 8 surrounding tiles. Use this to deduce safe spots.

3

Place Flags

Right-click (or long-press on mobile) to place a 🚩 flag on a tile you suspect is a mine. Flag all mines to win!

4

Win the Game

Reveal all safe tiles without hitting a mine. You win when every non-mine cell is uncovered.

Strategy Tips

  • → Start from a corner — corners have fewer neighbors, giving you more info early.
  • → If a 1 tile has exactly 1 hidden neighbor, that neighbor is definitely a mine.
  • → Flag confidently — the mine counter helps you track remaining mines.
  • → When stuck, count: if hidden neighbors = number shown, they're all mines.
  • → Empty (blank) reveals cascade automatically — use them to open large areas fast.
